# prompt-kit

prompt-kit is a developer toolkit for working productively with AI assistants like GitHub Copilot, Claude, and Gemini. It provides sensible instructions, structured prompts, and useful tools to streamline your workflow.

## Features

- Sensible, basic instructions for AI-assisted development
- Structured prompt templates for common tasks
- Tools for engaging with major AI assistants (Copilot, Claude, Gemini, and more)
- Designed for clarity, minimalism, and productivity

## Technology

- Built with TypeScript
- Planned support for major AI tools and platforms

## Usage

npx prompt-kit init

```

### CLI Usage

```

npx prompt-kit init

```
Prints `Initialized` and exits with code 0.

Any other command prints `Unknown command` and exits with code 1.
```
