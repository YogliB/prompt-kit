## ✨ What is Prompt Kit?

**Prompt Kit** is a CLI toolkit that scaffolds helper files for AI coding agents. Create structured templates, coding standards, and development configurations that enhance your AI-assisted development workflow.

**Get started in minutes** → **[📋 See Examples](./docs/EXAMPLES.md)**

## 🚀 Quick Start

### Installation

```bash
# Clone and setup
git clone https://github.com/YogliB/prompt-kit.git
cd prompt-kit
npm install

# Build and link globally (optional)
npm run build
npm link
```

### Basic Usage

```bash
# Show available commands
prompt-kit --help

# Check version
prompt-kit --version

# Custom greeting
prompt-kit [your-name]
```
